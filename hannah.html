History

This was the last version built of my portfolio, using Angular.js.

Although I loved building this site as it gave me more experience using Angular, ultimately I felt it was overkill for a one page site. The end result works great and looks wonderful, but I realized that I could have achieved the same result in this case with a simple JS and CSS solution- again reiterating my desire to use the right tool for the job.

I really enjoy working with Angular and look forward to using it more in projects that I build. You can see the end result here.
