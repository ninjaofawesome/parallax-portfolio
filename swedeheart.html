Swedeheart

Overview

History

Swedeheart was a project site that I built for my friend Cinna Liselott-Vesterberg as a way to build her puppetmaking party business. It was a good way for me to learn more about parallax scrolling as well as jQuery.

Cinna's site is also a high fidelity rapid prototype hosted on GitHub Pages with a custom domain attached, as it was the right build for her needs. I particularly enjoyed this project as it forced me to try new things as well as developing flexible, adaptable code.

You can check out the final result here, and perhaps make some of your own puppets!

Tools and Techniques Utilized In This Project

  • Bootstrap
  • Sass
  • jQuery
  • Parallax Scrolling
